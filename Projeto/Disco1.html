Menu Hamburger sem JavaScript

Este menu de navegação com três riscos (hamburger) foi criado usando apenas HTML e CSS, sem nenhum JavaScript.

Clique no ícone de três riscos no canto superior direito para abrir o menu.

Como funciona

Utilizamos um checkbox escondido e o seletor :checked do CSS para controlar a exibição do menu.

Vantagens

Não depende de JavaScript, é mais leve e funciona mesmo com JS desabilitado no navegador.

Responsivo

Este menu é totalmente responsivo e funciona perfeitamente em dispositivos móveis e desktop.
